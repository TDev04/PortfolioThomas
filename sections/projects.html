EZFCT Project

EasyFCT is a project designed to transform the management of training internships in Vocational Training (VET) centers. Its main objective is to overcome the limitations of current systems such as qBid and others, offering a modern, efficient and adaptable solution to the specific needs of each center. With an intuitive interface and advanced functionalities, EasyFCT enhances the user experience and facilitates the interaction between the different users of the Workplace Training (WTC) process, with its own API, and a complete custom bearer authentification service. Project developed during my studies between 2024 - 2025.

Internship Project

A Java-based application designed to manage employee attendance using NFC technology. The system integrates with an SQL database and provides two main interfaces: a simple user-facing screen for NFC-based check-in/out, and an administrative dashboard with secure login to manage users, view attendance logs, and update records. Additional features include automatic daily adjustments for incomplete check-outs and NFC device detection in real-time. Developed during 2024 - 2025.
